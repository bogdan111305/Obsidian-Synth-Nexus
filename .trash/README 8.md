# Part 2: Расширенная настройка Hibernate Envers

Этот раздел содержит материалы по расширенной настройке и кастомизации Hibernate Envers.

## Содержание раздела

### Основные темы

1. **[Интерфейс EnversListener](./Интерфейс EnversListener.md)** - Кастомизация процесса аудита
   - Интерфейс EnversListener и его методы
   - Создание кастомных слушателей
   - Интеграция с процессом аудита
   - Практические примеры использования

2. **[Классы AuditProcess и AuditWorkUnit](./Классы AuditProcess и AuditWorkUnit.md)** - Внутренняя работа Envers
   - Класс AuditProcess и его роль
   - Класс AuditWorkUnit и единицы работы
   - Взаимодействие между компонентами
   - Кастомизация процесса аудита

3. **[Custom revinfo table](./Custom revinfo table.md)** - Кастомизация таблицы ревизий
   - Создание кастомной таблицы ревизий
   - Аннотация @RevisionEntity
   - Кастомные поля в таблице ревизий
   - Интеграция с бизнес-логикой

4. **[Класс RevisionListener](./Класс RevisionListener.md)** - Слушатели ревизий
   - Интерфейс RevisionListener
   - Методы onRevisionCreate и onRevisionUpdate
   - Кастомизация информации о ревизиях
   - Практические сценарии использования

5. **[Продвинутые сценарии аудита](./Продвинутые сценарии аудита.md)** - Сложные случаи использования
   - Аудит коллекций и ассоциаций
   - Условный аудит
   - Аудит вложенных объектов
   - Производительность и оптимизация

### Ключевые концепции

- **EnversListener** - интерфейс для кастомизации процесса аудита
- **AuditProcess** - основной процесс обработки аудита
- **AuditWorkUnit** - единица работы аудита
- **RevisionEntity** - кастомная сущность для ревизий
- **RevisionListener** - слушатель событий ревизий
- **Custom revinfo** - кастомизация таблицы ревизий
