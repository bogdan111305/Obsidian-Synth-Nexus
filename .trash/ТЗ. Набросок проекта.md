## Описание

Web риложение для запоминания английских слов 
	- один модуль 
	- карточки слов, с ответами
	- аналитика по заученным словам
	- миниблог аналог твиттера, с заметками и прочему
	- составление собственного словаря, механизм возможности делится с другими участниками сообщества

## Технологии

	web: rest
	front: angular 2+
	back: 
		dev: springboot, postgres, gradle
		test: testContainers

#### Базовый функционал пользователя

Манипуляции с набором 

- создать набор + 
- добавить слова в набор + 
- удалить слова из набора + 
- удалить набор + 
- удалить набор у юзера + 
- посмотреть все слова набора  + 

*Пользователь может, создать набор и добавить в него слова. Посмотреть все слова набора. Удалить слова из набора. Удалить сам набор.*
