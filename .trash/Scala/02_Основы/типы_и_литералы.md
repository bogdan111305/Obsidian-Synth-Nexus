# Типы данных и литералы в Scala

## Обзор системы типов

Scala имеет богатую систему типов, которая объединяет объектно-ориентированные и функциональные концепции. Все типы являются объектами, включая примитивные типы.

## Базовые типы

### Числовые типы

```scala
// Целые числа
val byte: Byte = 127
val short: Short = 32767
val int: Int = 2147483647
val long: Long = 9223372036854775807L

// Числа с плавающей точкой
val float: Float = 3.14f
val double: Double = 3.14159265359

// Большие числа
val bigInt: BigInt = BigInt("123456789012345678901234567890")
val bigDecimal: BigDecimal = BigDecimal("3.141592653589793238462643383279")
```

### Символьный тип

```scala
val char: Char = 'A'
val unicodeChar: Char = '\u0041' // Unicode для 'A'
```

### Логический тип

```scala
val boolean: Boolean = true
val falseValue: Boolean = false
```

### Строковый тип

```scala
val string: String = "Hello, Scala!"
val multiLineString: String = """
  |Это многострочная
  |строка в Scala
  |""".stripMargin
```

## Литералы

### Целочисленные литералы

```scala
// Десятичные
val decimal = 42

// Шестнадцатеричные
val hex = 0xFF

// Восьмеричные (устарели в Scala 2.13+)
// val octal = 0777

// Двоичные
val binary = 0b101010

// С подчеркиваниями для читаемости
val readable = 1_000_000
```

### Литералы с плавающей точкой

```scala
val pi = 3.14159
val e = 2.71828f
val large = 1.23e-4
val veryLarge = 1.23E+10
```

### Символьные литералы

```scala
val letter = 'A'
val digit = '9'
val special = '\n' // Новая строка
val tab = '\t'     // Табуляция
val quote = '\''   // Одинарная кавычка
val backslash = '\\' // Обратная косая черта
```

### Строковые литералы

```scala
val simple = "Hello"
val withEscape = "Line 1\nLine 2"
val withQuote = "He said \"Hello\""
val withBackslash = "C:\\Users\\Name"

// Сырые строки
val raw = raw"C:\Users\Name" // Не обрабатывает escape-последовательности

// Интерполяция строк
val name = "Scala"
val greeting = s"Hello, $name!"
val expression = s"2 + 2 = ${2 + 2}"

// Форматирование
val formatted = f"Pi is approximately $pi%.2f"
```

### Многострочные строки

```scala
val multiLine = """
  |Это многострочная
  |строка с отступами
  |""".stripMargin

val sql = """
  |SELECT name, age
  |FROM users
  |WHERE age > 18
  |""".stripMargin
```

## Сложные типы

### Массивы

```scala
// Одномерные массивы
val numbers: Array[Int] = Array(1, 2, 3, 4, 5)
val strings = Array("a", "b", "c")

// Создание массива заданного размера
val emptyArray = new Array[Int](10)

// Доступ к элементам
val first = numbers(0)
numbers(0) = 10 // Изменение элемента

// Многомерные массивы
val matrix = Array.ofDim[Int](3, 3)
```

### Кортежи (Tuples)

```scala
// Кортежи разных размеров
val pair: (String, Int) = ("Alice", 25)
val triple: (String, Int, Boolean) = ("Bob", 30, true)

// Доступ к элементам
val name = pair._1
val age = pair._2

// Деструктуризация
val (firstName, userAge) = pair

// Вложенные кортежи
val nested: ((String, Int), Boolean) = (("Charlie", 35), false)
```

### Опциональные значения

```scala
val someValue: Option[String] = Some("Hello")
val noValue: Option[String] = None

// Работа с Option
val result = someValue.map(_.toUpperCase)
val default = noValue.getOrElse("Default")
```

## Выведение типов

Scala имеет мощную систему выведения типов:

```scala
// Тип выводится автоматически
val x = 42 // Int
val y = "Hello" // String
val z = List(1, 2, 3) // List[Int]

// Явное указание типа
val explicit: Double = 42
val list: List[String] = List("a", "b", "c")
```

## Типы функций

```scala
// Функция без параметров
val getTime: () => Long = () => System.currentTimeMillis()

// Функция с одним параметром
val double: Int => Int = x => x * 2

// Функция с несколькими параметрами
val add: (Int, Int) => Int = (a, b) => a + b

// Частично примененная функция
val addFive: Int => Int = add(5, _)
```

## Типы коллекций

```scala
// Списки
val list: List[Int] = List(1, 2, 3, 4, 5)

// Векторы
val vector: Vector[String] = Vector("a", "b", "c")

// Множества
val set: Set[Int] = Set(1, 2, 3, 4, 5)

// Отображения
val map: Map[String, Int] = Map("Alice" -> 25, "Bob" -> 30)

// Последовательности
val seq: Seq[Double] = Seq(1.1, 2.2, 3.3)
```

## Null и Nothing

```scala
// Null - подтип всех ссылочных типов
val nullString: String = null

// Nothing - подтип всех типов (нижняя граница)
def throwException(): Nothing = throw new RuntimeException("Error")

// Unit - тип для функций без возвращаемого значения
def printMessage(): Unit = println("Hello")
```

## Типы высшего порядка

```scala
// Функция, принимающая функцию как параметр
def applyFunction(f: Int => String, x: Int): String = f(x)

// Функция, возвращающая функцию
def createMultiplier(factor: Int): Int => Int = x => x * factor

// Использование
val double = createMultiplier(2)
val result = double(5) // 10
```

## Типичные ошибки

### 1. Неправильное использование типов

```scala
// Ошибка - несоответствие типов
val x: Int = "Hello" // Ошибка компиляции

// Правильно
val x: String = "Hello"
```

### 2. Работа с null

```scala
// Опасный код
val str: String = null
val length = str.length // NullPointerException

// Безопасный код
val str: Option[String] = Some("Hello")
val length = str.map(_.length).getOrElse(0)
```

### 3. Неправильное использование кортежей

```scala
// Ошибка - неправильный индекс
val pair = ("Alice", 25)
val name = pair._3 // Ошибка компиляции - только 2 элемента

// Правильно
val name = pair._1
```

## Лучшие практики

1. **Используйте выведение типов** для простых случаев
2. **Явно указывайте типы** для публичных API
3. **Предпочитайте иммутабельные типы**
4. **Используйте Option вместо null**

```scala
// Хорошо
def findUser(id: Int): Option[User] = {
  // Логика поиска
}

// Избегайте
def findUser(id: Int): User = {
  // Может вернуть null
}
```
