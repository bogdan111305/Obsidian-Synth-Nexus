# Part 1: Основы Envers

Этот раздел содержит базовые концепции Hibernate Envers и начальную настройку для аудита сущностей.

## Содержание раздела

### Основные темы

1. **[Подключение зависимости hibernate-envers](./Подключение зависимости hibernate-envers.md)** - Настройка Maven/Gradle
2. **[Аннотации @Audited и @NotAudited](./Аннотации @Audited и @NotAudited.md)** - Базовые аннотации аудита
3. **[Audit таблицы](./Audit таблицы.md)** - Структура и создание таблиц аудита

5. **[Базовые примеры использования](./Базовые примеры использования.md)** - Практические примеры

### Ключевые концепции

- **Аудит** - автоматическое отслеживание изменений в сущностях
- **Ревизии** - версии данных с временными метками
- **Audit таблицы** - таблицы для хранения истории изменений
- **@Audited** - аннотация для включения аудита
- **@NotAudited** - аннотация для исключения полей из аудита
