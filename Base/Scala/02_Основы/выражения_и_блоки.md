# Выражения и блоки в Scala

## Обзор

В Scala все является выражением, включая блоки кода. Это означает, что каждый блок кода возвращает значение, что делает язык более функциональным и выразительным.

## Выражения

### Простые выражения

```scala
// Числовые выражения
val sum = 1 + 2 + 3
val product = 4 * 5 * 6
val division = 10 / 3

// Строковые выражения
val greeting = "Hello" + " " + "World"
val interpolated = s"Sum is $sum"

// Логические выражения
val isPositive = 5 > 0
val isEven = 10 % 2 == 0
```

### Выражения с функциями

```scala
// Вызов функций как выражений
val length = "Hello".length
val upperCase = "hello".toUpperCase
val reversed = "Scala".reverse

// Цепочка вызовов
val result = List(1, 2, 3, 4, 5)
  .filter(_ % 2 == 0)
  .map(_ * 2)
  .sum
```

## Блоки кода

### Синтаксис блоков

Блок кода заключается в фигурные скобки `{}` и возвращает значение последнего выражения.

```scala
val result = {
  val x = 10
  val y = 20
  x + y // Это значение будет возвращено
}
// result = 30
```

### Блоки с побочными эффектами

```scala
val message = {
  println("Выполняется вычисление...")
  val temp = 42
  println(s"Результат: $temp")
  temp // Возвращаемое значение
}
```

### Вложенные блоки

```scala
val complexResult = {
  val outer = 100
  {
    val inner = 50
    outer + inner
  } * 2
}
// complexResult = 300
```

## Условные выражения

### if-выражения

В Scala `if` является выражением, а не оператором.

```scala
val x = 10
val result = if (x > 0) "positive" else "negative"

// Множественные условия
val grade = if (score >= 90) "A"
           else if (score >= 80) "B"
           else if (score >= 70) "C"
           else "F"
```

### Блоки в условных выражениях

```scala
val result = if (condition) {
  val temp = expensiveCalculation()
  temp * 2
} else {
  val default = getDefaultValue()
  default
}
```

## Циклы как выражения

### for-выражения

```scala
// for-выражение возвращает значение
val squares = for (i <- 1 to 5) yield i * i
// squares = Vector(1, 4, 9, 16, 25)

// С условием
val evenSquares = for (i <- 1 to 10 if i % 2 == 0) yield i * i
// evenSquares = Vector(4, 16, 36, 64, 100)
```

### while-циклы

```scala
// while возвращает Unit
val result = {
  var i = 0
  var sum = 0
  while (i < 10) {
    sum += i
    i += 1
  }
  sum // Явное возвращение значения
}
```

## Функции как выражения

### Лямбда-выражения

```scala
val add = (x: Int, y: Int) => x + y
val double = (x: Int) => x * 2
val isEven = (x: Int) => x % 2 == 0

// Использование
val result = add(5, 3) // 8
val doubled = double(10) // 20
```

### Функции высшего порядка

```scala
def applyFunction(f: Int => Int, x: Int): Int = f(x)

val result = applyFunction(x => x * x, 5) // 25
```

## Паттерн-матчинг как выражение

```scala
val value: Any = "Hello"

val result = value match {
  case s: String => s.length
  case i: Int => i * 2
  case _ => 0
}
```

## Try-выражения

```scala
val result = try {
  riskyOperation()
} catch {
  case e: IllegalArgumentException => -1
  case e: Exception => 0
} finally {
  cleanup()
}
```

## Блоки в определениях

### Определение функций

```scala
def calculateArea(radius: Double): Double = {
  val pi = 3.14159
  pi * radius * radius
}

// Однострочная функция
def square(x: Int): Int = x * x
```

### Определение классов

```scala
class Calculator {
  def add(x: Int, y: Int): Int = {
    val result = x + y
    println(s"$x + $y = $result")
    result
  }
}
```

## Ленивые вычисления

### lazy val

```scala
lazy val expensiveValue = {
  println("Вычисляется...")
  Thread.sleep(1000)
  42
}

// expensiveValue еще не вычислено
println("Начинаем...")

// Только сейчас вычисляется
val result = expensiveValue
```

## Блоки с областью видимости

```scala
val result = {
  val localVar = 100
  {
    val innerVar = 50
    localVar + innerVar
  }
}
// localVar и innerVar недоступны вне блоков
```

## Типичные ошибки

### 1. Забывание возвращаемого значения

```scala
// Ошибка - блок возвращает Unit
val result = {
  val x = 10
  val y = 20
  // Нет возвращаемого значения
}

// Правильно
val result = {
  val x = 10
  val y = 20
  x + y
}
```

### 2. Неправильное использование Unit

```scala
// Ошибка - ожидается Int, но получается Unit
val result: Int = {
  println("Hello")
  // Нет возвращаемого значения
}

// Правильно
val result: Int = {
  println("Hello")
  42
}
```

### 3. Смешивание выражений и операторов

```scala
// Ошибка - while возвращает Unit
val result = while (condition) {
  // действия
}

// Правильно - явное возвращение значения
val result = {
  var temp = 0
  while (condition) {
    temp += 1
  }
  temp
}
```

## Лучшие практики

### 1. Используйте выражения вместо операторов

```scala
// Хорошо
val result = if (condition) value1 else value2

// Избегайте
var result = 0
if (condition) {
  result = value1
} else {
  result = value2
}
```

### 2. Делайте блоки читаемыми

```scala
// Хорошо
val result = {
  val intermediate = calculateIntermediate()
  val processed = process(intermediate)
  finalize(processed)
}

// Избегайте слишком длинных блоков
val result = {
  val a = 1; val b = 2; val c = 3; val d = 4; val e = 5
  a + b + c + d + e
}
```

### 3. Используйте блоки для сложных вычислений

```scala
// Хорошо
val userInfo = {
  val user = findUser(id)
  val permissions = getUserPermissions(user)
  val settings = getUserSettings(user)
  UserInfo(user, permissions, settings)
}
```
