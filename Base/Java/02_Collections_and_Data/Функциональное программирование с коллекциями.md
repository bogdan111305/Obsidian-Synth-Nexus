# Функциональное программирование с коллекциями

> [!QUOTE] Суть
> Ключевые функциональные интерфейсы: `Function<T,R>` (T→R), `Consumer<T>` (T→void), `Predicate<T>` (T→boolean), `Supplier<T>` (→T), `BiFunction<T,U,R>`. `@FunctionalInterface` — интерфейс с ровно одним абстрактным методом. Лямбды — реализация через `invokedynamic`.

Функциональное программирование в Java предоставляет мощные инструменты для работы с коллекциями. Эта статья рассматривает лямбды, функциональные интерфейсы, Stream API и современные подходы к обработке данных.

## 1. Функциональные интерфейсы

### 1.1. Основные функциональные интерфейсы

```java
import java.util.function.*;

// Predicate<T> - проверка условия
Predicate<String> isLong = s -> s.length() > 10;
Predicate<Integer> isEven = n -> n % 2 == 0;

// Function<T, R> - преобразование
Function<String, Integer> getLength = String::length;
Function<Integer, String> toString = Object::toString;

// Consumer<T> - потребление значения
Consumer<String> printer = System.out::println;
Consumer<List<String>> sorter = Collections::sort;

// Supplier<T> - поставка значения
Supplier<Random> randomSupplier = Random::new;
Supplier<List<String>> listSupplier = ArrayList::new;

// BiFunction<T, U, R> - функция с двумя параметрами
BiFunction<String, String, String> concat = String::concat;
BiFunction<Integer, Integer, Integer> sum = Integer::sum;
```

### 1.2. Специализированные функциональные интерфейсы

```java
// Для примитивных типов
IntPredicate intPredicate = i -> i > 0;
LongFunction<String> longToString = String::valueOf;
DoubleConsumer doublePrinter = System.out::println;
IntSupplier randomInt = () -> new Random().nextInt();

// Бинарные операции
BinaryOperator<String> stringConcat = String::concat;
BinaryOperator<Integer> max = Integer::max;
BinaryOperator<List<String>> listMerge = (l1, l2) -> {
    l1.addAll(l2);
    return l1;
};
```

### 1.3. Создание собственных функциональных интерфейсов

```java
@FunctionalInterface
public interface TriFunction<T, U, V, R> {
    R apply(T t, U u, V v);
}

@FunctionalInterface
public interface Validator<T> {
    boolean isValid(T t);
    
    // Можно добавлять методы по умолчанию
    default Validator<T> and(Validator<T> other) {
        return t -> this.isValid(t) && other.isValid(t);
    }
    
    default Validator<T> or(Validator<T> other) {
        return t -> this.isValid(t) || other.isValid(t);
    }
}
```

### 2.2. Захват переменных

```java
public class LambdaCaptureExample {
    private String prefix = "Item: ";
    
    public void demonstrateCapture() {
        String localVar = "Local";
        int localNumber = 42;
        
        // Захват локальных переменных
        Consumer<String> printer = s -> {
            System.out.println(prefix + localVar + ": " + s + localNumber);
        };
        
        // Переменные должны быть effectively final

}
```

## 3. Stream API и функциональное программирование

### 3.1. Промежуточные операции

```java
List<String> data = Arrays.asList("apple", "banana", "cherry", "date");

// Фильтрация
Stream<String> filtered = data.stream()
    .filter(s -> s.startsWith("a"))
    .filter(s -> s.length() > 4);

// Преобразование
Stream<Integer> lengths = data.stream()
    .map(String::length);

// Плоское преобразование
Stream<Character> chars = data.stream()
    .flatMap(s -> s.chars().mapToObj(c -> (char) c));

// Сортировка
Stream<String> sorted = data.stream()
    .sorted();

// Удаление дубликатов
Stream<String> distinct = data.stream()
    .distinct();

// Ограничение
Stream<String> limited = data.stream()
    .limit(2);

// Пропуск элементов
Stream<String> skipped = data.stream()
    .skip(1);
```

### 3.2. Терминальные операции

```java
List<String> data = Arrays.asList("apple", "banana", "cherry");

// Сбор в коллекцию
List<String> list = data.stream()
    .collect(Collectors.toList());

Set<String> set = data.stream()
    .collect(Collectors.toSet());

// Подсчет
long count = data.stream().count();

// Поиск
Optional<String> first = data.stream().findFirst();
Optional<String> any = data.stream().findAny();

// Проверка условий
boolean allMatch = data.stream().allMatch(s -> s.length() > 3);
boolean anyMatch = data.stream().anyMatch(s -> s.startsWith("a"));
boolean noneMatch = data.stream().noneMatch(s -> s.length() > 10);

// Минимум и максимум
Optional<String> min = data.stream().min(String::compareTo);
Optional<String> max = data.stream().max(String::compareTo);

// Сведение к одному значению
Optional<String> reduced = data.stream()
    .reduce((a, b) -> a + ", " + b);
```

### 3.3. Специализированные коллекторы

```java
List<String> data = Arrays.asList("apple", "banana", "cherry", "date");

// Группировка
Map<Integer, List<String>> byLength = data.stream()
    .collect(Collectors.groupingBy(String::length));

// Разбиение
Map<Boolean, List<String>> partitioned = data.stream()
    .collect(Collectors.partitioningBy(s -> s.length() > 4));

// Подсчет
Map<String, Long> frequency = data.stream()
    .collect(Collectors.groupingBy(Function.identity(), Collectors.counting()));

// Соединение строк
String joined = data.stream()
    .collect(Collectors.joining(", "));

// Статистика
IntSummaryStatistics stats = data.stream()
    .mapToInt(String::length)
    .summaryStatistics();
```

## 4. Функциональные паттерны

### 4.1. Map-Reduce паттерн

```java
public class MapReduceExample {
    
    public static <T, R> R mapReduce(
            List<T> data,
            Function<T, R> mapper,
            BinaryOperator<R> reducer) {
        
        return data.stream()
            .map(mapper)
            .reduce(reducer)
            .orElseThrow(() -> new IllegalStateException("Empty data"));
    }
    
    public static void main(String[] args) {
        List<String> words = Arrays.asList("hello", "world", "java");
        
        // Подсчет общей длины всех слов
        Integer totalLength = mapReduce(
            words,
            String::length,
            Integer::sum
        );
        
        System.out.println("Total length: " + totalLength);
    }
}
```

### 4.2. Pipeline паттерн

```java
public class PipelineExample {
    
    public static <T> Pipeline<T> pipeline() {
        return new Pipeline<>();
    }
    
    public static class Pipeline<T> {
        private Stream<T> stream;
        
        public Pipeline() {
            this.stream = Stream.empty();
        }
        
        public Pipeline<T> filter(Predicate<T> predicate) {
            stream = stream.filter(predicate);
            return this;
        }
        
        public <R> Pipeline<R> map(Function<T, R> mapper) {
            stream = stream.map(mapper);
            return new Pipeline<>();
        }
        
        public List<T> collect() {
            return stream.collect(Collectors.toList());
        }
    }
}
```

### 4.3. Builder паттерн с функциональным подходом

```java
public class FunctionalBuilder<T> {
    private final List<Function<T, T>> operations = new ArrayList<>();
    
    public FunctionalBuilder<T> with(Function<T, T> operation) {
        operations.add(operation);
        return this;
    }
    
    public T build(T initial) {
        return operations.stream()
            .reduce(Function.identity(), Function::andThen)
            .apply(initial);
    }
    
    public static <T> FunctionalBuilder<T> builder() {
        return new FunctionalBuilder<>();
    }
}

// Использование
Person person = FunctionalBuilder.<Person>builder()
    .with(p -> p.setName("John"))
    .with(p -> p.setAge(30))
    .with(p -> p.setEmail("john@example.com"))
    .build(new Person());
```

## 5. Монады и функциональное программирование

### 5.1. Optional как монада

```java
public class OptionalMonadExample {
    
    public static Optional<String> processName(String name) {
        return Optional.ofNullable(name)
            .filter(n -> !n.trim().isEmpty())
            .map(String::toUpperCase);
    }
    
    public static Optional<Integer> parseAge(String ageStr) {
        try {
            return Optional.of(Integer.parseInt(ageStr));
        } catch (NumberFormatException e) {
            return Optional.empty();
        }
    }
    
    public static void demonstrateMonad() {
        // Цепочка операций с Optional
        Optional<String> result = Optional.of("john")
            .flatMap(this::processName)
            .flatMap(name -> Optional.of("Hello, " + name));
        
        // Обработка ошибок
        Optional<Integer> age = parseAge("25")
            .filter(a -> a > 0)
            .map(a -> a * 2);
    }
}
```

### 5.2. Stream как монада

```java
public class StreamMonadExample {
    
    public static <T, R> Stream<R> flatMap(Stream<T> stream, 
                                          Function<T, Stream<R>> mapper) {
        return stream.flatMap(mapper);
    }
    
    public static void demonstrateStreamMonad() {
        List<String> words = Arrays.asList("hello", "world");
        
        // flatMap для создания потока символов
        Stream<Character> chars = words.stream()
            .flatMap(word -> word.chars().mapToObj(c -> (char) c));
        
        // Цепочка операций
        List<String> result = words.stream()
            .flatMap(word -> Stream.of(word.toUpperCase(), word.toLowerCase()))
            .filter(word -> word.length() > 3)
            .collect(Collectors.toList());
    }
}
```

## 6. Функциональные утилиты

### 6.1. Функциональные композиции

```java
public class FunctionalUtils {
    
    // Композиция функций
    public static <A, B, C> Function<A, C> compose(
            Function<B, C> f, Function<A, B> g) {
        return a -> f.apply(g.apply(a));
    }
    
    // Частичное применение
    public static <T, U, R> Function<U, R> partial(
            BiFunction<T, U, R> f, T t) {
        return u -> f.apply(t, u);
    }
    
    // Каррирование
    public static <T, U, R> Function<T, Function<U, R>> curry(
            BiFunction<T, U, R> f) {
        return t -> u -> f.apply(t, u);
    }
    
    // Обратное каррирование
    public static <T, U, R> BiFunction<T, U, R> uncurry(
            Function<T, Function<U, R>> f) {
        return (t, u) -> f.apply(t).apply(u);
    }
}
```

### 6.2. Функциональные коллекторы

```java
public class FunctionalCollectors {
    
    // Коллектор для создания Map с группировкой
    public static <T, K, V> Collector<T, ?, Map<K, V>> toMap(
            Function<T, K> keyMapper,
            Function<T, V> valueMapper) {
        return Collectors.toMap(keyMapper, valueMapper);
    }
    
    // Коллектор для создания TreeMap
    public static <T, K, V> Collector<T, ?, TreeMap<K, V>> toTreeMap(
            Function<T, K> keyMapper,
            Function<T, V> valueMapper) {
        return Collectors.toMap(keyMapper, valueMapper, 
            (v1, v2) -> v2, TreeMap::new);
    }
    
    // Коллектор для создания ImmutableList
    public static <T> Collector<T, ?, List<T>> toImmutableList() {
        return Collectors.collectingAndThen(
            Collectors.toList(),
            Collections::unmodifiableList
        );
    }
}
```

## 7. Практические примеры

### 7.1. Обработка данных пользователей

```java
public class UserDataProcessor {
    
    public static class User {
        private String name;
        private int age;
        private String email;
        
        // Конструкторы, геттеры, сеттеры
    }
    
    public static List<String> getAdultUserEmails(List<User> users) {
        return users.stream()
            .filter(user -> user.getAge() >= 18)
            .map(User::getEmail)
            .filter(email -> email != null && !email.isEmpty())
            .collect(Collectors.toList());
    }
    
    public static Map<Integer, List<User>> groupUsersByAge(List<User> users) {
        return users.stream()
            .collect(Collectors.groupingBy(User::getAge));
    }
    
    public static Optional<User> findOldestUser(List<User> users) {
        return users.stream()
            .max(Comparator.comparing(User::getAge));
    }
    
    public static double getAverageAge(List<User> users) {
        return users.stream()
            .mapToInt(User::getAge)
            .average()
            .orElse(0.0);
    }
}
```

### 7.2. Функциональная валидация

```java
public class FunctionalValidator {
    
    public static <T> Validator<T> and(Validator<T>... validators) {
        return t -> Arrays.stream(validators)
            .allMatch(v -> v.isValid(t));
    }
    
    public static <T> Validator<T> or(Validator<T>... validators) {
        return t -> Arrays.stream(validators)
            .anyMatch(v -> v.isValid(t));
    }
    
    public static <T> Validator<T> not(Validator<T> validator) {
        return t -> !validator.isValid(t);
    }
    
    public static void validateUser(User user) {
        Validator<User> userValidator = and(
            user -> user.getName() != null && !user.getName().isEmpty(),
            user -> user.getAge() >= 0 && user.getAge() <= 150,
            user -> user.getEmail() != null && user.getEmail().contains("@")
        );
        
        if (!userValidator.isValid(user)) {
            throw new IllegalArgumentException("Invalid user data");
        }
    }
}
```
