# JSR - 250, 330

JSR (Java Specification Request) - спецификация, набор интерфейсов и аннотаций от JavaEE.
Spring предоставляет реализацию этих спецификаций.

## JSR 250 - Java Annotation

``` XML
jacarta-annotation-api
```

![[Pasted image 20220929225437.png]]

- Generated - помечаются сгенерированные классы
- ManageBean - аналог @Component 
- PostConstruct - после создания бина 
- PreDestroy - перед удалением бина 
- Priority - указать приоритетный бин
- Resource - inject зависимостей по имени
- Resources  - inject зависимостей по имени
