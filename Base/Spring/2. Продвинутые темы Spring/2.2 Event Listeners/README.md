- **ApplicationEvent** - базовый класс для событий Spring
- **ApplicationEventPublisher** - интерфейс для публикации событий
- **@EventListener** - аннотация для обработки событий
- **EventListenerMethodProcessor** - BeanFactoryPostProcessor для обработки @EventListener
