# Stateful и Stateless

Этот раздел содержит документацию по архитектурным подходам к управлению состоянием в веб-приложениях Spring.

## Содержание раздела

### Основные темы

1. **[Stateful и Stateless Beans.md](./Stateful и Stateless Beans.md)** - Архитектурные подходы
   - Stateful подход (сессии, память)
   - Stateless подход (токены, масштабируемость)
   - Сравнение подходов
   - Гибридные решения

### Ключевые концепции

- **Stateful** - сервер сохраняет состояние между запросами
- **Stateless** - сервер не сохраняет состояние, информация передается клиентом
- **Сессии** - механизм хранения состояния на сервере
- **JWT токены** - безсостоятельный механизм аутентификации
- **Масштабируемость** - способность системы расти
