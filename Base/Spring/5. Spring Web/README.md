# Spring Web

Этот раздел содержит документацию по веб-функциональности Spring Framework, включая Spring MVC, интерцепторы, веб-безопасность и архитектурные паттерны.

## Содержание раздела

### Основные темы

1. **[Interceptors](./Interceptors/README.md)** - Перехватчики HTTP запросов
   - HandlerInterceptor
   - WebMvcConfigurer
   - Жизненный цикл интерцептора
   - Практические примеры

2. **[Stateful и Stateless](./Stateful и Stateless/README.md)** - Архитектурные подходы
   - Stateful подход (сессии)
   - Stateless подход (токены)
   - Сравнение подходов
   - Гибридные решения

3. **[Spring MVC](./Spring MVC/README.md)** - Основы веб-фреймворка
   - Контроллеры
   - View Resolvers
   - Data Binding
   - Validation

4. **[REST API](./REST API/README.md)** - RESTful веб-сервисы
   - @RestController
   - RequestMapping
   - ResponseBody
   - HTTP методы

### Ключевые концепции

- **HandlerInterceptor** - перехват HTTP запросов
- **Stateful/Stateless** - управление состоянием приложения
- **Spring MVC** - веб-фреймворк
- **REST API** - RESTful сервисы
- **Web Security** - веб-безопасность
