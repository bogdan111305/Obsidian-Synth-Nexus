# Конфигурация Spring

## Содержание раздела

### Способы конфигурации
- **XML-based конфигурация** - классический подход
- **Annotation-based конфигурация** - современный подход
- **Java-based конфигурация** - программная конфигурация
- **Properties и YAML** - внешняя конфигурация

### Файлы конфигурации
- **application.properties** - основные настройки
- **application.yml** - YAML формат
- **application-{profile}.properties** - профили

### Аннотации конфигурации
- **@Configuration** - класс конфигурации
- **@Bean** - определение бинов
- **@ComponentScan** - сканирование компонентов
- **@PropertySource** - источники свойств
