# Аннотации Spring

## Содержание раздела

### Основные аннотации
- **@Component** - базовая аннотация для создания бинов
- **@Service** - для сервисного слоя
- **@Repository** - для слоя доступа к данным
- **@Controller** - для веб-контроллеров

### Аннотации для Dependency Injection
- **@Autowired** - автоматическое внедрение зависимостей
- **@Value** - внедрение значений из properties
- **@Resource** - JSR-250 аннотация для DI
- **@Qualifier** - уточнение конкретного бина

### Аннотации конфигурации
- **@Configuration** - класс конфигурации
- **@Bean** - определение бина в конфигурации
- **@ComponentScan** - сканирование компонентов

### Условные аннотации
- **@Conditional** - условное создание бинов
- **@Profile** - профили приложения
