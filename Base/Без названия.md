Подключиться к Postgres можно прямо через Docker:

  docker exec -it agg-postgres psql -U postgres -d arenadata

  Дальше обычный SQL. Несколько готовых примеров:

  Создать задачу:
  INSERT INTO utm.ut (id, code, name, status, process_id, trace_id, due_date, context, created_at, updated_at)
  VALUES (
    gen_random_uuid(),
    'TSK-MY-001',
    'Моя тестовая задача',
    'CREATED',
    'proc-1',
    gen_random_uuid(),
    NOW() + INTERVAL '7 days',
    '{}',
    NOW(), NOW()
  );

  Добавить ACL:
  INSERT INTO utm.ut_acl (id, ut_id, grant_type, grant_value, perms, updated_at)
  VALUES (gen_random_uuid(), '<ut_id>', 'user', 'user-42', ARRAY['READ','WRITE'], NOW());

  Изменить статус:
  UPDATE utm.ut SET status='COMPLETING', assignee='user-99', updated_at=NOW()
  WHERE code='TSK-MY-001';

  ---
  Смотреть что происходит — три места:

  Логи индексера (видно когда Kafka Streams эмитит документ):
  docker logs -f utm-cdc-indexer

  OpenSearch — проверить документ по id:
  docker exec agg-opensearch curl -s \
    "http://localhost:9200/utm-ut-index/_doc/utm:ut:<ut_id>" | python3 -m json.tool

  Kafka UI — браузер, топики и сообщения в реальном времени:
  http://localhost:8090

  Задержка от INSERT до появления в OpenSearch обычно 3–8 секунд (Debezium → Kafka → Streams commit interval 5s → Sink коннектор).
